# 🎯 AI Interview Coach

An AI-powered interview preparation platform that simulates real interview experiences using LLMs. The system generates role-based questions, evaluates answers, and provides structured feedback to help users improve their interview performance.

---

## 🚀 Features

- 🎯 Generates technical, HR, and scenario-based interview questions
- 🧠 AI-powered answer evaluation using Google Gemini API
- 📊 Structured feedback (clarity, depth, technical understanding)
- ✍️ Improved answer suggestions from AI
- 🛤️ Learning roadmap generator for career roles
- 💾 Saves interview history using Firebase Firestore
- 💻 Clean and interactive Streamlit UI

---

## 🧠 How It Works

1. User selects a job role (DS / ML / Software Engineer)
2. AI generates realistic interview questions
3. User writes an answer
4. AI evaluates response using structured scoring
5. Feedback + improved answer is generated
6. Results are stored in Firebase for tracking progress
